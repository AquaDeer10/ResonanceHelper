## 使用说明

### 模拟器

推荐使用mumu模拟器

若使用其他模拟器，请修改 ```adb.ini``` 中的 ```device_port``` 为对应模拟器的adb端口

分辨率请设为 ```1920 * 1080```

### 游戏设置

分辨率请尽量设置为 ```高``` 或 ```极高```

设置为```中``` ```低``` 可能会影响识别准确率

### 下载预构建版本

[release](https://github.com/AquaDeer10/ResonanceHelper/releases)

### 自行构建

#### 环境准备

- OS: Windows
- Python < 3.12

#### 构建

```
.\build.cmd
```
